# 📊 Production Sales & Production Analysis Dashboard

## 📌 Project Overview
This project analyzes sales, production, and operational efficiency for a garments factory. The goal is to identify key business insights such as top buyers, product performance, production gaps, and delivery efficiency.

## 📁 Dataset
The dataset includes:

* **Order ID**
* **Buyer**
* **Product Type**
* **Order Quantity**
* **Production Quantity**
* **Defects**
* **Delivery Date**
* **Actual Shipment Date**
* **Price per Unit**
* **Sales**

## 🎯 Key Analysis Performed

### 1. Top Buyers Analysis
* Identified top buyers based on total sales
* Calculated sales contribution (%)
* **Insight:** Revenue is evenly distributed across buyers, reducing dependency risk

### 2. Top Products Analysis
* Analyzed best-performing products
* Compared product performance based on sales and quantity

### 3. Monthly Sales Trend
* Aggregated daily data into monthly view
* Identified trends and seasonality

### 4. Order vs Production Analysis
* Compared ordered vs produced quantity
* Calculated Production Gap
* **Insight:** Production consistently falls short of demand across buyers

### 5. Defect Analysis
* Measured defect count and defect percentage
* Identified which buyers/products contribute most to defects

### 6. On-Time Delivery Rate
* Calculated using: `On-Time = Actual Shipment Date ≤ Delivery Date`
* Evaluated delivery performance across buyers

## 📊 Tools Used
* Microsoft Excel (Excel Online)
* Pivot Tables
* Charts (Bar, Line, Combo)
* Slicers
* Formulas (IF, COUNTIF, TEXT, etc.)



## 💡 Key Insights
* Sales are well distributed across buyers (~18–21%), reducing business risk
* Production is consistently lower than order quantity, indicating supply constraints
* Certain product categories contribute more to defects
* Delivery performance can be improved for operational efficiency

## 🚀 How to Use
1. Download the Excel file
2. Open in Microsoft Excel (recommended for full functionality)
3. Use slicers to interact with the dashboard
